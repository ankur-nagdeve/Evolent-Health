# Contact Management Portal

Application for maintaining contact information.

## Getting Started

Following is tech-stack of this application –

* ASP.NET Web API
* MVC 5
* Database SQL Server 2012
* Entity Framework (Database First Approach)
* AngularJS Version 1.5


### Steps to run 


```
* Using SSMS, restore the database ‘ContactDB’ from the backup file provided in the repository.

* Open ContactManagement.sln file in visual studio.Open ContactMgmtModel.edmx file under Models folder

* Right click, use ‘Update the Model from database’ option. Set up new connection to connect to ‘ContactDB’ database created in step1 above.

* Run the application, you should be presented with Dashboard page.
```
